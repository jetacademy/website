# Style Conventions
This project follows these style and coding conventions:

## Laravel/PHP Backend
- PSR-12 coding standard.
- Parameterized queries using Eloquent ORM to prevent SQL injection.
- Eager loading (`with()`) to prevent N+1 query issues.

## Frontend (Vue.js)
- Options API or Composition API as established in components.
- Single File Components (.vue).
- Mobile-first, responsive, and unique UI/UX styling.
